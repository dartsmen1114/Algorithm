# 19. 删除链表的倒数第 N 个结点
[力扣原题传送门](https://leetcode-cn.com/problems/remove-nth-node-from-end-of-list/)

### 解题思路
这道题如果直接暴力破解的话，就是先从头到尾的遍历一遍，拿到链表的长度后在，计算倒数第n个节点的下标位置
然后删除该节点，这样做的话可能会超时，所以换个思路。</p>
1.首先找到第n个节点所在的位置，这个应该不难。
