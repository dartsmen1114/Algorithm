> 跟着《算法小抄》走，首先，你要熟悉二叉树，要对前序遍历，中序遍历，后序遍历了如指掌，几乎所有的算法都是从二叉树的遍历演变而来。
> 套用一句经典的话：如果你告诉我，快速排序就是个二叉树的前序遍历，归并排序就是个二叉树的后续遍历，那么我就知道你是个算法高手了。
