> 如果你和我一样从来没刷过题，那我建议你看看《算法小抄》，首先，你要熟悉二叉树，要对前序遍历，中序遍历，后序遍历了如指掌，
> 大多数复杂的算法都是从从二叉树的遍历演变而来。例如快速排序就是个二叉树的前序遍历，归并排序就是个二叉树的后续遍历。因此
> 刷题的话还是要先刷二叉树，循序渐进。
